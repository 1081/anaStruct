StructuralEngineering 2D Frames and Trusses (Raamwerken en vakwerken)
=====================================================================

This is a small package for python 3. With it you can solve 2D Frames
and Trusses calculations. Determine the bending moment, shear force,
normal force and the displacements.

See the wiki for some examples.
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

`Read the docs! <https://github.com/ritchie46/structural_engineering/wiki>`_

Installation
------------

::

    $ pip install StructuralEngineering

For the latest version download this repository on github and unzip the file. Open
the command box in the unzipped file location and run:

::

    $ python setup.py install


See the github link:

`Github repository structural_enineering <https://github.com/ritchie46/structural_engineering>`_
